# Centro de Control AVA Pro v9 Donut Ejecutivo

Versión con sidebar tornasol, ícono de autobús y mejora en la gráfica de recuperación:

- Porcentaje de recuperación en el centro del donut.
- Resumen lateral: cobrado, adeudo y total AVA.
- Colores degradados más profesionales.
- Mantiene lectura desde Google Apps Script para TRT, SUR, AVATRT y AVASUR.
