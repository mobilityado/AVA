# Centro de Inteligencia AVA v20

Plataforma ejecutiva AVA con módulos:

- Dashboard Ejecutivo: KPIs, recuperación, adeudos y pronóstico.
- Conductores: historial, búsqueda y panel de detalle.
- Cajeros: ranking, recuperación y desempeño.
- Rutas: origen-destino más concurrido y análisis.
- Tendencias: mensual, anual y comparativos.
- Resumen Inteligente: insights automáticos para gerencia.
- Reportes: exportación PDF y Excel/CSV.
- Configuración: metas, umbrales, modo claro/oscuro y auto actualización.

## Publicación en GitHub Pages

Sube todo el contenido del ZIP al repositorio y activa GitHub Pages en `main / root`.

Después abre la página con `Ctrl + F5` para evitar caché.
