# CIO AVA Enterprise 3.0

Versión Decision Intelligence basada en CIO AVA Enterprise 2.5 y en la lógica estable sin duplicidad:

- AVATRT y AVASUR: KPIs, incidencias, tendencias, porcentajes y reportes generales.
- TRT y SUR: ranking y detalle de cajeros exclusivamente.

## Novedades

- Executive Score de 0 a 100.
- Recomendaciones automáticas basadas en reglas de negocio.
- Escenarios de predicción conservador, esperado y optimista.
- Mapa operativo de riesgo por origen-destino.
- Línea del tiempo de eventos.
- Reconocimientos automáticos para cajeros y diploma PDF.
- Centro de alertas en el encabezado.
- CIO Copilot 3.0 con memoria de contexto local.
- PWA actualizada a Enterprise 3.0.

## Instalación

1. Sube todo el contenido a la raíz del repositorio de GitHub Pages.
2. Mantén las carpetas `assets`, `css` y `js`.
3. No es necesario actualizar `Code.gs` si Enterprise 2.5 ya funciona.
4. Desregistra el Service Worker anterior o abre una ventana privada la primera vez.
5. Actualiza con Ctrl + F5.

El motor de inteligencia funciona en el navegador y no envía datos a servicios externos.
