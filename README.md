# CIO AVA v42 — Identidad, PWA y reportes corporativos

## Incluye
- Logotipo oficial SVG escalable: autobús, gráfica ascendente y escudo.
- Pantalla de inicio animada con la identidad CIO AVA.
- Iconos SVG personalizados para todos los módulos.
- Plantilla de PDF corporativa con nombre CIO AVA y recuperación general.
- PWA instalable en Windows, Android y navegadores compatibles.
- Iconos 192×192, 512×512 y Apple Touch.
- Service Worker para cargar la interfaz aun con conexión inestable.

## Publicación en GitHub Pages
1. Sube **todo el contenido** de esta carpeta a la raíz del repositorio.
2. Conserva las carpetas `assets`, `css` y `js`.
3. Espera a que GitHub Pages publique los cambios.
4. Abre con `Ctrl + F5` una vez.
5. Para instalar: pulsa **Instalar app** cuando aparezca en el encabezado o usa la opción de instalación del navegador.

> La consulta de datos requiere conexión porque proviene de Google Apps Script. La PWA conserva disponible la interfaz y archivos estáticos.
