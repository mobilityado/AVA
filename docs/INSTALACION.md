# Instalación

1. Sustituir `Code.gs` en Google Apps Script únicamente si la instalación actual no corresponde a la versión v47/v48 con usuarios y auditoría.
2. Publicar una nueva versión de la aplicación web y conservar la URL `/exec` configurada en el proyecto.
3. Subir todo el contenido de esta carpeta a la raíz del repositorio de GitHub Pages.
4. Conservar las carpetas `assets`, `css`, `js` y `docs`.
5. Limpiar el Service Worker anterior o abrir con `Ctrl + F5`.
6. Probar primero en incógnito y después reinstalar la PWA si aplica.
